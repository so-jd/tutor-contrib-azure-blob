azure-blob plugin for `Tutor <https://docs.tutor.edly.io>`__
############################################################

Store assests on Azure blob


Installation
************

.. code-block:: bash

    pip install git+https://github.com/so-jd/tutor-contrib-azure-blob

Usage
*****

.. code-block:: bash

    tutor plugins enable azure-blob
